# C# Software Development Knowledge Base

本项目旨在系统地记录从 C# 基础语法到工业级 WPF 软件开发的完整知识体系。

## 目录结构

### [01_Basics](./01_Basics) - 语言基础
- [基础语法](./01_Basics/basic-syntax.md)
- [控制流](./01_Basics/control-flow.md)
- [面向对象编程](./01_Basics/object-oriented-programming.md)

### [02_Environment](./02_Environment) - 开发环境
- [.NET Framework 概览](./02_Environment/dotnet-framework-overview.md)
- [.NET CLI 基础](./02_Environment/cli-tools/dotnet-cli-basics.md)

### [03_Desktop_UI](./03_Desktop_UI) - 桌面开发 (WPF)
- `xaml-fundamentals/` - XAML 基础与布局
- `mvvm-basics/` - MVVM 模式入门
- `custom-controls/` - 自定义控件开发
- `animations-and-styling/` - 动画、资源字典与皮肤

### [04_Software_Architecture](./04_Software_Architecture) - 软件架构
- `dependency-injection/` - 依赖注入 (DI)
- `service-layer/` - 服务层设计
- `design-patterns/` - 常用设计模式 (单例, 工厂, 观察者等)

### [05_External_Integration](./05_External_Integration) - 外部集成
- `python-interop/` - C# 调用 Python 脚本 (Process/StandardInput)
- `database-sql/` - PostgreSQL/Dapper/Entity Framework
- `docker-devops/` - Docker 容器化与环境隔离
- `ai-model-deployment/` - YOLO 模型集成与推理

### [06_Engineering_Standards](./06_Engineering_Standards) - 工程标准
- `algorithms/` - 算法设计与优化
- `concurrency-async/` - 多线程与 async/await
- `memory-management/` - 内存管理与性能优化 (GC, IDisposable)
- `cicd-pipeline/` - CI/CD 自动化构建

### [07_Practical_WpfApp](./07_Practical_WpfApp) - 实战案例
- 记录 `Blade Inspector` 项目中的具体技术实现与避坑指南。
